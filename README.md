# Titanic Survival Prediction using Logistic Regression
📌 Aim

The aim of this project is to analyze the Titanic dataset and build a predictive model to determine passenger survival outcomes based on demographic and travel-related features. Through exploratory data analysis, the project identifies key factors such as age, gender, passenger class, and fare that influence survival probability. It also focuses on handling missing data, transforming categorical variables, and preparing the dataset for machine learning. Using Logistic Regression, the project develops a classification model that predicts survival likelihood while evaluating performance through metrics like accuracy, precision, recall, F1-score, and ROC-AUC. The objective is to demonstrate how data preprocessing and statistical modeling can be applied to solve real-world classification problems.

## Project Overview

This project applies end-to-end data analysis and machine learning techniques on the Titanic dataset. It includes:

Data loading and initial exploration
Descriptive statistics and distribution analysis
Visualization of target variable (Survival)
Feature analysis (Gender, Age, etc.)
Missing value handling and data cleaning
Feature engineering and One-Hot Encoding
Train-test split and data standardization
Logistic Regression model training
Model evaluation using confusion matrix and classification report
ROC curve and AUC score analysis

## Key Insights
Gender plays a major role: females had higher survival rates
Passenger class significantly impacts survival probability
Age distribution shows variation in survival chances
Fare and class combined influence survival likelihood
The model achieved ~81% accuracy with strong recall for survivors
